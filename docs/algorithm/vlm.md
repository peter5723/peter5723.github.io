# 多模态

项目名称：基于 AIGC 合成数据与多模态大模型的零样本目标检测流水线

AIGC 全合成数据生成（冷启动攻坚）：针对特定军装/军帽等涉密或稀缺场景“零真实数据”的冷启动痛点，设计基于 Stable Diffusion 的合成数据工厂。通过精细化 Prompt 工程与参数调优（结合复杂环境、光照、遮挡等长尾扰动），零成本规模化生成 4,000+ 张高逼真度的场景图像，从源头解决下游模型过拟合与数据采集成本极高的问题。

VLM 自动化标注与底层并发管线：彻底抛弃手工打标，引入 Grounding DINO 多模态大模型对 SD 生成的图像进行“即插即用”的自动化包围框（BBox）提取。开发基于 ThreadPoolExecutor 的高并发预处理异步流，解耦图像解码与 GPU 推理，在严控显存 OOM 的前提下将吞吐量提升 300%，40 分钟内完成全量数据的标签对齐与格式转换。

Teacher-Student 伪标签管线与端侧部署：构建了“大模型指导小模型”的数据闭环。将 Grounding DINO（Teacher）产出的高质量硬标签（Hard Labels）直接作为伪标签，投喂给端侧网络 YOLOv8n（Student）。成功将多模态大模型的零样本泛化能力迁移至百兆以内的轻量级模型中，在未见过的真实测试集上实现了高可用 mAP，同时将单图推理延迟从大模型的 400ms+ 压缩至 5ms 级别，完成了从数据合成到端侧实时部署的全链路落地。
